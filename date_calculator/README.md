# Date Calculator - Frontend

This is the Next.js frontend application for the Date Calculator project.

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the application.

## Build

```bash
npm run build
npm start
```

## Environment Variables

- `NEXT_PUBLIC_API_URL`: API URL (if using an external API service)

For more information about the full project, see the [main README](../README.md).
